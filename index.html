Cyc-Sort

When people throw away their diposables, they often have difficulty determining what is recyclable and what is not. Due to this, many default to throwing their disposables in the trash. Even worse, sometimes trash ends up in recycling, which can cause everything else in the recycling bin to be disposed as trash. To encourage more recycling and to make sure it's done right, Cyc-Sort uses a motor controlled platform and computer vision technology to automatically sort people's disposables.

How Does Cyc-Sort Work?

Take a look at the components that make Cyc-Sort tick.

Sensors

Ultrasonic Sensor

The ultrasonic sensor on the platform detects when an object is placed in Cyc-Sort, which signals Cyc-Sort to start classifying the object.

​

Pi Camera

Once Cyc-Sort gets a signal from the ultrasonice sensor, it uses the Pi camera to take a picture and determine if the object is recyclable or not.

​

IR Distance Sensor

Cyc-Sort uses IR Distance Sensors above each compartment to determine the capacity of each bin; if a bin is too full then the lid will close and remain closed until the bin is cleared.

​

Actuators

Lid Servo

Cyc-Sort uses a servo to open and close the lid, which made of living wood.

​

Platform Servo

After Cyc-Sort classifies the object, it uses the servo on the platform to place the object in the correct bin by rotating clockwise or counter-clockwise.

Implementation Challenges

The infrared distance sensors that we used has analog output, but our raspberry pi only has digital input; so we had to use an Arduino Nano board to make up for that lack of functionality.

Since the raspberry pi is unable to provide enough power for multiple servos on its own, we had to use external batteries to run our servos. However, the batteries were not reliable enough so we ended up using a dedicated power supply instead.
